# 🛒 Amazon Clone

This is a frontend Amazon clone built using React. It replicates core e-commerce functionalities like cart management and checkout.

---

## 🚀 Features

* 🛍️ Add to Cart
* ❌ Remove from Cart
* 💳 Checkout Page
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* ⚛️ React JS
* 🌐 Context API
* 🎨 CSS

---

## ▶️ Run Locally

```bash
npm install
npm start
```
